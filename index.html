The economics of utopia

Isabel Almudí, Francisco Fatás, Luis R. Izquierdo & Jason Potts

HOW TO INSTALL THE MODEL

To use Utopia, you will have to install NetLogo 5.3.1 (free and open source) and download the model itself. Unzip the downloaded file and click on utopia.nlogo

DESCRIPTION OF THE MODEL

Utopia is described in the paper "The economics of utopia: A co-evolutionary model of ideas, citizenship and socio-political change".

LICENCE

Utopia is an evolutionary model designed to explore the coevolution of ideas in a socio-political system.
